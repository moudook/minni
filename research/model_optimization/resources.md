# Resources: Model Optimization

## Graph Optimization
- [ONNX Runtime Graph Optimizations](https://onnxruntime.ai/docs/performance/graph-optimizations.html) - Operator fusion guide.
- [TensorFlow Graph Transform Tool](https://github.com/tensorflow/tensorflow/blob/master/tensorflow/tools/graph_transforms/README.md) - Legacy but useful reference.

- [Torch Pruning](https://pytorch.org/tutorials/intermediate/pruning_tutorial.html) - Official PyTorch tutorial.
- [Hugging Face Distillation Guide](https://huggingface.co/docs/transformers/en/tasks/knowledge_distillation) - Practical guide for Transformers.
