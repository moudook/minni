# Resources: CPU-Only Solutions

## Architecture & Instruction Sets
- [ARM NEON Programmer's Guide](https://developer.arm.com/documentation/den0018/a/) - Official guide for SIMD on ARM.
- [ARM Architecture Reference Manual](https://developer.arm.com/documentation/ddi0487/latest) - Detailed instruction set reference.

## Libraries & Kernels
- [XNNPACK](https://github.com/google/XNNPACK) - High-efficiency floating-point neural network inference operators.
- [Ruy](https://github.com/google/ruy) - Matrix multiplication library used in TFLite.
- [ARM Compute Library](https://developer.arm.com/Tools%20and%20Software/Compute%20Library) - Optimized low-level functions.

## Threading & Optimization
- [Android Performance Patterns](https://www.youtube.com/playlist?list=PLWz5rJ2EKKc9CBxr3BVjPTPoDPLdPIF_c) - Google's guide to mobile performance.
- [Gemmlowp](https://github.com/google/gemmlowp) - Low-precision matrix multiplication.

## Articles/Tutorials
- [Optimizing Matrix Multiplication](https://github.com/flame/how-to-optimize-gemm) - A guide to high-performance C.
