# Resources: Benchmarking

## Tools
- [Android Profiler](https://developer.android.com/studio/profile/android-profiler) - Official IDE tool.
- [Simpleperf](https://developer.android.com/ndk/guides/simpleperf) - Native sampling profiler.
- [Perfetto](https://perfetto.dev/) - System-wide tracing.

## Libraries
- [Google Benchmark](https://github.com/google/benchmark) - Microbenchmark library for C++.
- [JankStats](https://developer.android.com/topic/performance/jankstats) - UI performance tracking.
