# Examples

How to use the framework.

## Structure
```
examples/
├── vision_examples/        # Image examples
├── language_examples/      # Text examples
├── audio_examples/         # Sound examples
├── optimization_examples/  # Making things faster
└── android_examples/       # Android specific
```

## Rules
- Must work on CPU only phones
- Add comments to explain code
- Show basic and advanced ways
- Handle different phone capabilities
- Show how to deal with errors
