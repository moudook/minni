# Code Style Guide

## Kotlin (Android)
- Follow official [Android Kotlin Style Guide](https://developer.android.com/kotlin/style-guide).
- Use `kdocs` for all public methods.

## C++ (Core)
- Follow [Google C++ Style Guide](https://google.github.io/styleguide/cppguide.html).
- Use `clang-format`.

## Python (Optimization)
- Follow PEP 8.
- Use `black` for formatting.
