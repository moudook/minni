# Getting Started

## Prerequisites
- Android Studio Koala (2024.1.1) or newer.
- Android NDK r26+.
- Python 3.10+ (for model conversion).

## Installation
1. Clone the repo.
2. Open `android/` in Android Studio.
3. Sync Gradle.

## Running Your First Model
1. Connect a device (Developer Mode enabled).
2. Run the `examples/android_demo` app.
3. Select "MobileNet V2" from the dropdown.
